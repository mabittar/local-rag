# Documento de Teste

## Introdução

Este é um documento Markdown de teste para verificar o processamento de arquivos.

## Seções

### Upload

O sistema deve permitir upload de arquivos em múltiplos formatos:
- PDF
- TXT
- DOCX
- MD

### Processamento

Após o upload:
1. Extração de texto
2. Chunking
3. Geração de embeddings
4. Armazenamento no PostgreSQL

## Conclusão

Este documento serve para testar o fluxo completo de ingestão.
